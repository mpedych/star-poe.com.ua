== Description ==

Добавления способа оплаты [LiqPay](https://www.liqpay.ua/api/referral/i13901439870) - v.3 для WooCommerce.
В этой версии плагина покупатели смогут только оплачивать товары из корзины вашего интернет магазина выбрав способ оплаты LiqPay. 
Но в этой версии нет callback на ваш сайт после оплаты. Сallback - это обращения к вашему сайту по API с сервиса liqpay для уведомления Вас, что деньги поступили на счет и тем самым изменяет статус в ваших заказах в админ панели на "Обработка" .  
Для более полной версии плагина с callback вы можете заказать написав на Email: pavel.borysenko@gmail.com
При покупки полной версии плагина, в целях безопасности продажи плагина я даю номер карточки Приват Банка на имя Борисенко Павел Сергеевич. Если во время покупки вам дают карточку другого банка или на другое имя, то не оплачивайте, так как это могут быть мошенники.

== Установка ==

1. Разархивируйте содержимое zip файла в папку плагинов вашего сайта (wp-content/plugins/), используя вашу любимую FTP программу.
3. Активируйте плагин на странице "Плагины" в панели администратора.
4. Всё установка завершена.

После установки и активации плагина зайдите в админке в левом меню в WooCommerce -> Настройки (Settings) -> Платежи (Payments) и активируйте LiqPay, затем зайдите в него и пропишите public_key и private_key - их вы получите при регистрации в системе liqpay.ua и добавления в него вашего сайта.
